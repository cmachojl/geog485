Question 2

What are the required parameters, and what do they represent?

The required parameters are

SERVICE=WMS, which represents service type and REQUEST=GetCapabilities which represents request name.

What is returned in response to a WMS GetCapabilities request?

In a GetCapabilities request Service Metadata is returned which is an XML file representing information about a specific WMS service and its component layers.

Question 3

What are the required parameters, and what do they represent?

Version=version, REQUEST=map, REQUEST=GetMap, LAYERS=layers_list, STYLES=style_list, SRS-namespace identifier, BBOX=minx, miny, maxx, maxy, Width=output_width, Height=output_height, FORMAT=output_format

What is returned in response to a WMS GetMap request?

The response that is returned from a GetMap request are graphic files representing one or more layers from a single WMS service for a specified area of interest, and optionally, for a specified point in time.

What is the significance of transparency in WMS requests?
